## Retrieving the code
To download the contents of this repository, go to the top right of this page and click on "Code". Then, under "Download source code", click on "zip". 
This downloads the repository as a ZIP file. Locate the ZIP file in your downloads folder, unzip it and you are ready to go!

## Running the code
Before running the code, we first have to install the dependencies locally. 
For this, follow the following steps:
1. Navigate to the Cases ExquAIro folder
2. Open a git bash terminal via right click -> Open Git Bash here 
3. In the Git Bash terminal type: ```uv sync```
This will create a virtual environment and install all required packages. 

4. Open VSCode and in VSCode open the Cases ExquAIro folder (top left click File -> Open Folder...)
5. Open the case files from case 1 or 2 in the corresponding 'notebooks' folders, and click on 'Select Kernel' (top right) to select the .venv we just created
  - When doing this for the first time, click on the pop-up 'Install/Enable suggested extenstions', and wait until installation finishes. If done before, this step is not required.
  - The pop-up 'Python Environments' will show, click and select the .venv (Python 3.8.20)

When these steps are completed, you should be able to run the notebook! For each new notebook, please select the .venv as kernel.

Enjoy!
